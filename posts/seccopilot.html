🪙 Security Copilot clever nutzen

Original veröffentlicht auf LinkedIn

🪙
Security Copilot wär cool – aber mein Konto sagt: „Arrr, bitte nicht!“

🧢 Herausforderung:

Du willst Microsoft Security Copilot nutzen – aber 24/7 für rund 4.000 € ist dir zu happig?
Keine Sorge, Captain Azure hat einen cleveren Workaround für dich, ganz ohne Piraterie:

🛠️ Der Trick: Stundenweise einsetzen

  • Erstelle eine eigene Resource Group → z. B. rg-copilot-on-demand
  • Bau dir einen Power Automate Flow mit dem Schritt:
    👉 Create or update a template deployment

Pack da nur diese paar Zeilen rein:

{
  "$schema": "https://lnkd.in/eEuzYGCW",
  "contentVersion": "1.0.0.0",
  "parameters": {},
  "variables": {},
  "resources": [
    {
      "type": "Microsoft.SecurityCopilot/capacities",
      "apiVersion": "2023-12-01-preview",
      "name": "sfseccopilotdev",
      "location": "westeurope",
      "properties": {
        "numberOfUnits": "1",
        "crossGeoCompute": "NotAllowed",
        "geo": "EU"
      }
    }
  ]
}

Den Flow teilst du z. B. in einem Teams-Kanal:

  • → Jeder, der Security Copilot braucht, drückt auf den Button – und los geht’s
  • → Mach das Deployment entweder inkrementell oder prüfe vorher, ob die RG schon existiert

Löschen nicht vergessen!

  • → Entweder automatisch per Zeitplan
  • → Oder mit einem zweiten „Abriss“-Flow (Risk: „Mach ich später…“ 🫣)

💡 Ergebnis?

  • Copilot auf Abruf
  • Nur zahlen, wenn aktiv genutzt
  • Kein Dauervertrag
  • Ideal für punktuelle Analysen, Audits, Krisen

🧭 Captain’s Fazit:

Warum ein ganzes Monatsabo zahlen,
wenn du auch auf Stundenbasis navigieren kannst?

Ok, Controller hassen diesen Trick – außer sie arbeiten bei dir 😇

Mit Azure + ein bisschen Automatisierung kannst du Security Copilot ganz flexibel andocken – budgetfreundlich und einsatzbereit, wenn’s wirklich drauf ankommt.
